# GeoPoint Resolver

Resolve localidades para coordenadas geográficas e distribui múltiplos pontos usando a espiral de Fermat — o mesmo algoritmo usado pelo sistema `geopoint` (`supergeomarker.py`).

## Quando usar

- Usuário tem um CSV com localidades (cidade, estado, país) e precisa de coordenadas
- Precisa preencher/corrigir uma coluna `GEOLOCALIZAÇÃO` em um CSV existente
- Quer gerar pontos distribuídos dentro de uma região (espiral de Fermat)
- Mencionou "geopoint", "geolocalizar", "coordenadas", "lat/lon"
- Precisa resolver pendências de geocodificação que APIs externas falharam

## Fluxo de trabalho

### 1. Ler o CSV e identificar pendências

Leia o CSV com a ferramenta Read. Identifique:
- Quais colunas contêm cidade, estado, país (busque padrões: "Cidade", "Municipio", "Estado", "UF", "País")
- Se já existe coluna `GEOLOCALIZAÇÃO` — se sim, identifique linhas vazias
- Quantas linhas precisam de coordenadas, agrupadas por localidade

### 2. Resolver coordenadas

Use seu conhecimento geográfico para determinar as coordenadas centrais de cada localidade. Você conhece as coordenadas de praticamente qualquer lugar do mundo — cidades, aldeias, terras indígenas, ilhas, distritos, países.

Regras para geocodificação:
- **Cidades/municípios**: use o centroide da área urbana
- **Países (quando o CSV lista artistas de outros países)**: use a capital como ponto de referência
- **Terras indígenas / aldeias**: use a coordenada mais conhecida ou o centroide aproximado
- **Ilhas**: use o centroide geográfico da ilha
- **Localidades ambíguas**: priorize o contexto do CSV (se as outras linhas são do Brasil, "Colômbia" pode ser um município brasileiro — mas se a coluna Estado diz "Colômbia (País)", é o país)
- **`nan` ou vazio**: ignore, não tente geocodificar

Para cada localidade, defina:
- `lat`: latitude do centro
- `lon`: longitude do centro  
- `radius_km`: raio aproximado para distribuição dos pontos (default: 15km para cidades, 50km para estados/regiões grandes, 5km para aldeias/distritos pequenos)

### 3. Gerar pontos com espiral de Fermat

O script `scripts/fermat_geocode.py` implementa o algoritmo. O fluxo:

1. Crie um arquivo JSON temporário com as coordenadas resolvidas:
```json
{
  "Manaus - Amazonas": {"lat": -3.1190, "lon": -60.0217, "radius_km": 20},
  "Belém - Pará": {"lat": -1.4558, "lon": -48.5024, "radius_km": 18},
  "Estados Unidos": {"lat": 38.8951, "lon": -77.0364, "radius_km": 50}
}
```

2. Execute o script:
```bash
python3 ~/.claude/skills/geopoint-resolver/scripts/fermat_geocode.py \
  "<csv_entrada>" "<csv_saida>" "<json_coordenadas>"
```

3. O script:
   - Lê o CSV de entrada
   - Identifica linhas sem `GEOLOCALIZAÇÃO`
   - Agrupa por localidade (cidade + estado)
   - Filtra linhas com `nan`
   - Para cada grupo, gera N pontos em espiral de Fermat usando as coordenadas do JSON
   - Salva o CSV de saída com a coluna preenchida
   - Retorna JSON com estatísticas

### 4. Validar resultado

Após executar o script, leia o CSV de saída e confirme:
- Quantas linhas foram preenchidas
- Se ainda restam linhas vazias (e por quê — geralmente linhas sem cidade/estado)
- Mostre um resumo ao usuário

## Algoritmo da espiral de Fermat

O algoritmo distribui N pontos de forma uniforme e visualmente agradável dentro de um raio circular ao redor do centroide:

- **Ângulo dourado**: `pi * (3 - sqrt(5))` — garante distribuição uniforme sem clusters
- **Raio crescente**: `r_max * sqrt(i / n)` — pontos mais densos no centro, espalhando para fora
- **Correção de longitude**: divide o deslocamento X por `cos(latitude)` para compensar a projeção equiretangular (sem isso, pontos ficam esticados perto dos polos)
- **Precisão**: 7 casas decimais (~ 1cm de precisão)

Quando há apenas 1 ponto, retorna exatamente o centroide. Quando há muitos pontos (>50), considere aumentar o `radius_km`.

## Formato de saída

A coluna `GEOLOCALIZAÇÃO` usa o formato `latitude,longitude` (sem espaço):
```
-3.1190275,-60.0217314
```

## Exemplo de uso completo

Usuário: "Preencha as coordenadas que faltam nesse CSV"

1. Leia o CSV, identifique 15 localidades pendentes
2. Resolva as coordenadas de cada uma usando seu conhecimento
3. Crie o JSON temporário em `/tmp/coords.json`
4. Execute `fermat_geocode.py input.csv output.csv /tmp/coords.json`
5. Confirme que 15 localidades foram resolvidas
6. Mostre resumo ao usuário
