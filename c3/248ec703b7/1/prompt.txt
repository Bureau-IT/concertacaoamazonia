## Passo 3 — Corrigir arquivo-do-estudo (PDFs)

O WP importer remapeia _thumbnail_id automaticamente, mas `arquivo-do-estudo` usa IDs de attachment que existem em prod com IDs diferentes dos do HML.

```bash
ssh concertacaoamazonia.com.br-prod-sa "cd /var/www/concertacaoamazonia.com.br && sudo -u www-data wp db query \"
SELECT ID, guid FROM wp_posts WHERE post_type='attachment' AND (
  guid LIKE '%SC_PORT_Seguranca-Climatica-da-Amazonia.pdf%'
  OR guid LIKE '%IDB-IDB-Invest-Private-Sector-Roadmap%pdf%'
  OR guid LIKE '%Youth-Manifest.pdf%'
  OR guid LIKE '%cartografias-violencia-amazonia-2025.pdf%'
  OR guid LIKE '%The-Amazon-Report-2025.pdf%'
  OR guid LIKE '%Radar-Educacional-Amazonico.pdf%'
  OR guid LIKE '%Amazonia_plano_diretor_municipal_e_gestao_do_territorio_2017.pdf%'
  OR guid LIKE '%A-Amazonia-Diante-do-Seu-Futuro-Volume-1.pdf%'
  OR guid LIKE '%A-Amazonia-Diante-do-Seu-Futuro-Volume-2.pdf%'
  OR guid LIKE '%Cartilha-para-liderancas-e-multiplicadoras.pdf%'
  OR guid LIKE '%Cartilha-para-organizacoes.pdf%'
) ORDER BY ID\""
```

Com os IDs retornados, executar UPDATEs em `wp_postmeta` para `meta_key='arquivo-do-estudo'`
em posts 89232, 89238, 89245, 89252, 89256, 89631, 89651, 89707, 89715, 89721, 89727
(mesma lógica do local — os IDs de destino virão da query acima).
