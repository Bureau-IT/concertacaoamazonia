# Reorganização da Estrutura de Cores CSS

## Objetivo

Reorganizar as variáveis CSS no arquivo `style.css` em uma estrutura hierárquica clara e flexível, preparada para dark mode, onde funcionalidades e elementos visuais referenciam a paleta básica por importância.

## Estrutura Proposta

### 1. Paleta de Cores Básica (Brand)

Agrupar todas as cores principais do brand com hierarquia clara:

- **Primária** (`--brand-primary`): Cor principal (#900042)
- **Secundária** (`--brand-secondary`): Cor secundária (#DEDDD1) 
- **Terciária** (`--brand-tertiary`): Cor de destaque/accent (#F0C400)
- **Quaternária** (`--brand-quaternary`): Cor adicional (#C02975)
- Variações de cada cor (light, dark, hover states)
- Cores neutras (white, black, offwhite)

### 2. Cores Extras

Segundo bloco com até 4 variações extras:

- `--color-extra-1` até `--color-extra-4`
- Manter as existentes e organizá-las

### 3. Funcionalidades (Grandes Áreas)

Cores específicas para grandes áreas do site que **referenciam** a paleta básica por importância:

- **Header**: usa primária para background, terciária para hover
- **Eventos**: usa primária para textos, secundária para backgrounds
- **Formulários**: usa primária para textos, secundária para borders
- **Paginação**: usa primária/quaternária conforme contexto
- **Navegação**: usa primária/terciária
- **Espiral**: usa cores do brand conforme ciclo
- **Breaknews**: usa cores do brand

### 4. Elementos Visuais

Elementos de UI que também referenciam a paleta básica:

- **Botões**: normal, invertido, slider banner (referenciam primária/secundária/terciária)
- **Tags**: referenciam primária/neutras
- **Bordas**: referenciam cores do brand
- **Focus/Outline**: referenciam terciária/quaternária
- **Scrollbar**: referenciam terciária

### 5. Preparação para Dark Mode

Estrutura que permite dark mode facilmente:

- Variáveis base no `:root`
- Sobrescrita em `[data-theme="dark"]` ou `@media (prefers-color-scheme: dark)`
- Funcionalidades e elementos visuais continuam funcionando pois referenciam variáveis base
