### Passo 3: Ativar página de manutenção no ALB (Fixed Response)

Obter ARN do listener HTTPS do ALB:
```bash
ALB_ARN=$(AWS_PROFILE="Concertação" aws elbv2 describe-load-balancers \
  --names amazonia-alb --region sa-east-1 \
  --query 'LoadBalancers[0].LoadBalancerArn' --output text)

LISTENER_ARN=$(AWS_PROFILE="Concertação" aws elbv2 describe-listeners \
  --load-balancer-arn "$ALB_ARN" --region sa-east-1 \
  --query 'Listeners[?Port==`443`].ListenerArn' --output text)
```

Criar regra de manutenção (priority 1, acima de todas as outras):
```bash
AWS_PROFILE="Concertação" aws elbv2 create-rule \
  --listener-arn "$LISTENER_ARN" \
  --region sa-east-1 \
  --priority 1 \
  --conditions '[{"Field":"path-pattern","PathPatternConfig":{"Values":["/*"]}}]' \
  --actions '[{
    "Type": "fixed-response",
    "FixedResponseConfig": {
      "StatusCode": "503",
      "ContentType": "text/html",
      "MessageBody": "<HTML_MINIFICADO_AQUI>"
    }
  }]'
```
